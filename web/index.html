About

The ScribeCat Dev Console provides quick status checks for connectivity, the bundled static server, and the desktop app shell.

Use the hotkeys below to streamline routine checks and theme adjustments:

  • T toggle theme
  • R rerun status checks
  • L clear log entries
  • ? view shortcut help
Press ? for shortcuts.

Keyboard shortcuts

  • ? Toggle this overlay
  • T Toggle theme
  • R Rerun status checks
  • L Clear recent logs

Shortcuts are ignored while typing in inputs or editable regions.
